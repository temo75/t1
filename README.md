# ტურისტული ბილეთების პლატფორმა

ეს არის სატესტო ვებ-საიტი, რომელიც განკუთვნილია
ქართველი ტურისტებისთვის.

მომავალში შესაძლებელი იქნება:
- მგზავრობის ბილეთების ნახვა
- ფასების შედარება
- ონლაინ შეძენა
- მრავალენოვანი მხარდაჭერა (ქართული, ინგლისური, ბერძნული)

პროექტი დამწყების მიერ სწავლების მიზნით.
